Using on the web

You need to copy the entire UIIconFont directory into your project folder. In the <head> of your html, reference the location to your infogram/css/style.css.

Here’s an example:

<link rel="stylesheet" href="myProjects/webProject/infogram/css/style.css">

Example

< i class="infogram-airplane"> </i>
< i class=" infogram-building1"> </i>
< i class=" infogram-chart1"> </i>
